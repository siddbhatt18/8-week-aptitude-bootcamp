**Topic: Basic Work & Efficiency Concepts**

**Core Principles:**
```
Fundamental Concepts:
- Work = Rate √ó Time
- Rate (Efficiency) = Work/Time
- If A completes work in n days:
  * A's 1 day work = 1/n
  * A's efficiency = 1/n units/day

Key Relations:
- More efficient = Less time
- Efficiency ‚àù 1/Time
- If A is x times efficient as B:
  * Time_B = x √ó Time_A

Units Method:
- Total Work = LCM of all individual times
- This avoids fractions

Important Points:
- Always work with per day rates
- Combined efficiency = Sum of individual efficiencies
- Work done = Efficiency √ó Days worked
```

**Quick Reference Formulas:**
```
1. (A+B)'s 1 day work = A's 1 day + B's 1 day
2. If A does in x days, B in y days:
   Together = xy/(x+y) days
3. Efficiency ratio = Inverse of time ratio
4. Man-days = Men √ó Days = Constant
```

**Topic: Pipes & Cisterns Fundamentals**

**Core Concepts:**
```
Basic Principles:
- Inlet pipe = Positive work (fills)
- Outlet pipe = Negative work (empties)
- Net rate = Inlet rate - Outlet rate

Key Formulas:
- If pipe fills in x hours: Rate = 1/x per hour
- If pipe empties in y hours: Rate = -1/y per hour
- Combined rate = Œ£(inlet rates) - Œ£(outlet rates)

Special Cases:
1. Leak in tank (continuous outlet)
2. Multiple pipes with different capacities
3. Pipes opened/closed at intervals
4. Partial filling requirements

Conversion Note:
- All Time & Work concepts apply
- Replace "work" with "tank capacity"
- Efficiency = Filling/Emptying rate
```

#### **9:30 - 10:30 AM: Practice Problems**

**Problem Set 4 (Basic Pipes):**

4. Three pipes A, B, C fill in 12, 15, 20 hrs. All opened. B closed after 2 hrs. Total time?

   - Time for remaining = (2/3)/(2/15) = 5 hrs
   - Total = 7 hours

5. Pipe A fills in 10 hrs, B empties in 15 hrs. Opened alternately starting with A. Time?
   - 2-hour cycle: 1/10 - 1/15 = 1/30
