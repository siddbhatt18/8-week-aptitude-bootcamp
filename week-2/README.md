### üåÖ **Morning Session (1.5 hours)**

#### **9:00 - 9:30 AM: Concept Learning**
**Topic: Fundamental P&L Concepts**

**Core Terminology & Formulas:**
```
Basic Terms:
- CP = Cost Price (buying price)
- SP = Selling Price (selling price)
- Profit = SP - CP (when SP > CP)
- Loss = CP - SP (when CP > SP)

Percentage Formulas:
- Profit% = (Profit/CP) √ó 100
- Loss% = (Loss/CP) √ó 100
- SP = CP √ó (1 + Profit%/100)
- SP = CP √ó (1 - Loss%/100)
- CP = SP √ó 100/(100 + Profit%)
- CP = SP √ó 100/(100 - Loss%)

Quick Relations:
- 25% profit means SP = 1.25 √ó CP
- 20% loss means SP = 0.80 √ó CP
```
